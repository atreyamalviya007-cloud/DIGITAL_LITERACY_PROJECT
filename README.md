## 🔐 TASK 5: Phishing Awareness & Cyber Safety

### 🚨 What is Phishing?
A cyberattack where attackers trick users into revealing sensitive information via fake emails, messages, or websites.

### ⚙️ How It Works:
1. Fake message from a trusted source  
2. Urgent call to action  
3. Redirect to fake website  
4. Data is stolen  

### ⚠️ Risks:
- Financial loss  
- Identity theft  
- Account hacking  

### 🛡️ Prevention Checklist:
- Verify sender email  
- Never share OTP/passwords  
- Use strong passwords  
- Enable 2FA  
- Avoid suspicious links  
- Check URLs carefully  
- Avoid public Wi-Fi for transactions  
- Keep apps updated  

### 💳 UPI Safety:
- Never approve unknown requests  
- Verify receiver details  
- Report fraud at **cybercrime.gov.in**  
- Call **1930** in case of fraud  

### 💡 Reflection:
- Phishing attacks can look very real  
- Always verify before taking action  
- Awareness is key to prevention
